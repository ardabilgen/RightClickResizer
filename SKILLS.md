# Agent Skills & Methodology

This project has been enhanced using skills and methodologies from the following ecosystems:

## 1. Skills from [skills.sh](https://skills.sh/)
- **`python-testing-patterns`**: Implemented `pytest` based unit tests for configuration and image resizing logic.
- **`python-packaging`**: Introduced `pyproject.toml` for modern project metadata and dependency management.
- **`python-error-handling`**: Improved exception handling in registry operations and image processing.

## 2. Methodology from [superpowers](https://github.com/obra/superpowers)
- **`test-driven-development`**: New features are verified via tests before implementation.
- **`systematic-debugging`**: A 4-phase root cause process is used for troubleshooting context menu installation issues.
- **`writing-plans`**: Work is broken down into bite-sized, verifiable tasks.

## Installation
To use these skills in your own AI agents, you can install them via:
```bash
npx skills add wshobson/agents/python-testing-patterns
npx skills add obra/superpowers
```
